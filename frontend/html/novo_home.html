Check Bus

Bem-vindo ao Check Bus!

Este site foi criado para facilitar o acompanhamento dos horários, localização do ônibus e acesso aos relatórios dos alunos.

  • Editar Horários: Gerencie seus horários de embarque e desembarque.
  • Abrir GPS: Veja a localização do ônibus em tempo real.
  • Relatórios: Consulte relatórios de presença e uso.
  • Lista de Alunos: Administre os dados dos alunos cadastrados.

Use o menu lateral para navegar entre as funcionalidades do sistema.
